Module 08 — Q1: SQL vs NoSQL

Question
How do you choose between SQL and NoSQL for a service?

Answer
- Consider data model, query patterns, consistency needs, and operational constraints. SQL for relational integrity and complex queries; NoSQL for flexible schema and high-scale access patterns.

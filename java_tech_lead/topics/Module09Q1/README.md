Module 09 — Q1: Securing microservice APIs (input validation)

Question
How do you secure a microservice API? (short focus: input validation)

Answer
- Use TLS, authenticate and authorize requests, validate inputs to prevent injection attacks, and apply rate limiting and least privilege for secrets.

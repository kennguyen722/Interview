Q1: How do you approach technical trade-offs in a cross-functional team?
Answer: Clarify goals and constraints, propose options with pros/cons, quantify impact, solicit feedback from stakeholders, and recommend a path with rollback plan. Emphasize outcomes and team alignment.

Q2: How to mentor engineers and raise team technical bar?
Answer: Run regular code reviews focused on learning, set up pair-programming sessions, define standards, create learning goals, and provide regular feedback.

Q3: How to handle an on-call incident where the team disagrees on the root cause?
Answer: Focus first on mitigation and restoring service, gather data for postmortem, document decisions, and run a blameless postmortem to converge on root cause and action items.

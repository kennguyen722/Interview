01-java / 01-basics

This folder covers Java Basics prompts such as:
- Prompt 1: JDK vs JRE vs JVM
- Prompt 2: Primitive vs Reference types
- Prompt 3: String immutability and String pool
- Prompt 4: Wrapper classes and autoboxing
- Prompt 5: Java memory areas (stack vs heap)

Tasks
- Read the prompt description in the course pack.
- Work through the examples in `examples/`.
- Use `ExerciseTemplate.java` from `../..../templates/` to create your own practice files.

- `Example_Prompt1_JDK_JRE_JVM.java` — compile & run example demonstrating javac/java
- `Example_Prompt2_PrimitiveVsReference.java` — shows primitives vs references
- `Example_Prompt3_StringImmutability.java` — demonstrates String immutability and pool
- `Example_Prompt4_WrapperClasses.java` — autoboxing/unboxing examples
- `Example_Prompt5_StackVsHeap.java` — memory allocation demonstration

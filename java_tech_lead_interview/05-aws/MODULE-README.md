# Module 05: AWS Cloud Services

Cloud-native development patterns for Java Tech Leads leveraging core AWS services.

## Learning Objectives
- Provision and manage compute (EC2, Lambda, ECS/Fargate)
- Store and retrieve objects & block/file data (S3, EBS, EFS)
- Build messaging workflows (SQS, SNS)
- Design persistence with RDS & DynamoDB
- Secure workloads using IAM best practices
- Integrate API Gateway & networking (VPC fundamentals)
- Monitor and trace with CloudWatch & X-Ray
- Apply Terraform / CloudFormation snippets for IaC

## Submodules
- 01-compute
- 02-storage
- 03-messaging
- 04-database
- 05-networking
- 06-security
- 07-monitoring

## Estimated Duration
2-3 weeks

## Prerequisites
- Basic cloud concepts
- Java build & dependency management

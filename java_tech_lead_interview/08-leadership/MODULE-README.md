# Module 08: Technical Leadership

Develop people, processes and strategic communication skills required of a Java Tech Lead.

## Learning Objectives
- Build, mentor and grow high-performing teams
- Delegate effectively with clarity & accountability
- Handle performance management & feedback loops
- Communicate with stakeholders using structured frameworks
- Resolve conflicts through principled approaches
- Apply decision frameworks & risk mitigation techniques

## Submodules
- 01-team-management
- 02-communication
- 03-conflict-resolution
- 04-decision-making

## Estimated Duration
2-3 weeks

## Prerequisites
- Senior engineering experience recommended

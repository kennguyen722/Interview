# Module 03: Spring Framework

Enterprise Java development mastery using Spring Core, Boot, Data, Security, Testing, Cloud & Reactive.

## Learning Objectives
- Master IoC, DI and bean lifecycle
- Build RESTful APIs with Spring MVC
- Persist data using Spring Data JPA & transactions
- Secure applications with Spring Security (auth, authz, OAuth2/JWT)
- Apply Spring Boot auto-configuration, actuator, profiles
- Implement caching, messaging, and cloud patterns (Config, Discovery, Circuit Breakers)
- Write effective unit, slice and integration tests
- Explore reactive programming with WebFlux

## Submodules
- 01-core
- 02-mvc
- 03-data
- 04-boot
- 05-security
- 06-testing
- 07-cloud
- 08-reactive

## Estimated Duration
4-5 weeks

## Prerequisites
- Solid Java & OOP
- Basic build tool (Maven/Gradle)
